<h1>辞典オンライン</h1>
<div id="maininnerall">
<div id="maininner">
<p class="waku01-b">辞典オンラインは漢字・四字熟語・故事ことわざなどの辞書サイトです。</p>
<div class="sitelist">
<div class="sitelist1"> <a href="http://kanji.jitenon.jp/"><img src="img/site1.gif" width="202" height="52" alt="漢字辞典オンライン"></a>
<p>漢字の「部首」「画数」「読み方」「意味」（常用漢字には筆順）などを調べることができます。<br>
<a href="http://kanji.jitenon.jp/">→漢字辞典オンライン</a></p>
</div>
<div class="sitelist2"> <a href="http://yoji.jitenon.jp/"><img src="img/site2.gif" width="202" height="52" alt="四字熟語辞典オンライン"></a>
<p>四字熟語の「読み方」「意味」「出典」などを調べることができます。<br>
<a href="http://yoji.jitenon.jp/">→四字熟語辞典オンライン</a></p>
</div>
</div>
<div class="sitelist">
<div class="sitelist1"> <a href="http://kotowaza.jitenon.jp/"><img src="img/site3.gif" width="202" height="52" alt="故事ことわざ辞典オンライン"></a>
<p>故事やことわざの「読み方」「意味」「出典」などを調べることができます。<br>
<a href="http://kotowaza.jitenon.jp/">→故事・ことわざ辞典オンライン</a></p>
</div>
<div class="sitelist2"> <a href="http://chimei.jitenon.jp/"><img src="img/site4.gif" width="202" height="52" alt="地名辞典オンライン"></a>
<p>全国の地名を「都道府県」「市町村」のほか「近い読み」や「同じ漢字を含む地名」などから調べることができます。<br>
<a href="http://chimei.jitenon.jp/">→地名辞典オンライン</a></p>
