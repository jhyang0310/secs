</header>
<div class="entry-content">
<h2>271.0</h2>
<ul>
<li>Game.ini [ShooterGameMode] bUseTameLimitForStructuresOnly config option, which if true will make Tame Units only be applied and used for Platforms with Structures and Rafts. Defaults false. <li>?bRawSockets server option now has correct performance (server stalls eliminated), as good or better than Steam sockets</li>
<li>Fixed some boss exploits.</li>
</ul>
<h2>271.1</h2>
<ul>
<li>Baby Bone Scalings &amp; Gestation Setups now stack from Mods, and Dino Entry Icon now stack from Mods.</li>
<li>Fixed client side dino initialization crash.</li>
<li>Fixed with issue with local events on client.</li>
</ul>

<header class="entry-header">
<h2><a href="http://www.ark-survival.net/en/2017/09/18/ark-survival-evolved-patch-269-1/">ARK: Survival Evolved Patch 269.1 &#038; 269.2</a></h2>
</header>
<div class="entry-content">
<h2 style="text-align: justify;">269.1</h2>
<ul style="text-align: justify;">
<li>Client-side update to fix &#8220;Direct3D Device Lost&#8221; errors.</li>
</ul>
<h2 style="text-align: justify;">269.2</h2>
<h2 style="text-align: justify;">Changes</h2>
<ul style="text-align: justify;">
<li>Reduced Flamethrower Damage from base 100 per second to 30 per second.</li>
<li>Flamethrower Direct Damage is now affected by Armour.</li>
<li>Flamethrower expensive foliage particle effects now eliminated for everyone except the local player using the weapon</li>
<li>Chainsaw not able to attack whilst running</li>
<li>Tripled the effective &#8220;ammo&#8221; of TEK armours, so they last 3x as long per element</li>
<li>Shield blocks should block whip stuns/item drops (and trigger the cooldown), and whip stun on players reduced to 1 second.</li>
<li>Implemented a new value to reduce indirect torpor (Gas grenades, tranq).</li>
<li>Chitin, Metal, Riot, and TEK Armours now reduce incoming torpor.
<ul>
<li>Chitin:
<ul>
<li>Direct Torpor Armor Mult: 0.07 (Was 0)</li>
<li>Indirect Torpor Armor Mult: 0.035 (Was 0)</li>
</ul>
</li>
<li>Metal:
<ul>
<li>Direct Torpor Armor Mult: 0.15 (Was 0)</li>
<li>Indirect Torpor Armor Mult: 0.06 (Was 0)</li>
</ul>
</li>
<li> Riot:
<ul>
<li>Direct Torpor Armor Mult: 0.25 (Was 0.25)</li>
<li>Indirect Torpor Armor Mult: 0.12 (Was 0)</li>
</ul>
</li>
<li> Tek (remember there are no Tek qualities):
<ul>
<li>Direct Torpor Armor Mult: 0.7 (Was 0)</li>
<li>Indirect Torpor Armor Mult: 0.7 (Was 0)</li>
</ul>
</li>
</ul>
</li>
</ul>
<h2 style="text-align: justify;">Fixes &amp; Improvements</h2>
<ul>
<li style="text-align: justify;">Fixed a bug where in the event of a rollback, players who recently used the CrossARK transfer system could lose their characters.</li>
<li style="text-align: justify;">Fixed the bear trap to avoid turret exploit. Bear traps only damageable by explosion now (or demolishing)</li>
</ul>
</div>
